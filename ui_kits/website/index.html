# Getting started with forge

> A thirty-second install and a day of use.

Forge is a single CLI that speaks to multiple AI coding assistants behind one interface. You install it once, point it at a repo, and stop remembering which tool expects which flag.

## Install

npx forge-ai-assist@latest --plugins core

The core plugin set is read-only — Forge will analyse your repository, answer questions, and propose patches, but it won't write to disk until you explicitly let it.

## First session

Run forge inside a git-clean directory. You'll see the prompt drop into an interactive session with claude-code by default. Pass --assistant codex to swap it.

The catalog keeps itself honest, so if you follow a link that lands here, please let us know.

## What's next

Read the forge project page for the full feature matrix, or jump to github.com/arcanelabsio/forge.
